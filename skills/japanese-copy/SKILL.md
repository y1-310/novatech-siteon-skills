# japanese-copy — 日本語コピー品質ガイド

このスキルはNovaTech/SateOnの日本語テキスト生成に関する共通ルールです。
マスターファイル: `_common/japanese-copy-guide.md`

## 主要ルール（要約）

### 文体レベル（最重要）

| 業態 | レベル | 文末例 |
|------|:-----:|--------|
| lp-salon / lp-nail-esthe | **A（丁寧・上品）** | 〜しています / 〜いただけます |
| lp-barber / lp-restaurant | **B（カジュアル）** | 〜します / 〜できます |
| lp-architect / lp-corporate | **C（信頼・格式）** | 〜いたします / 〜しております |

- 1セクション内で文末の敬語レベルを混在させない
- 見出し（体言止め）と本文（ですます）の混在はOK

### 文字数制限

| 要素 | 上限 |
|------|:----:|
| 見出し h1/h2 | 20文字 |
| サブ見出し h3 | 30文字 |
| 本文1文 | 60文字 |
| CTA文言 | 10文字 |
| alt属性 | 30文字 |

### 禁止カタカナ（全業態）

ソリューション / オプティマイズ / コンバージョン / エンゲージメント / プラットフォーム / アジャイル / イノベーション / シナジー / エクスペリエンス / パーソナライズ / エンパワー / ジャーニー

### 英語直訳禁止パターン

| 禁止 | 置き換え |
|------|---------|
| 〜を次のレベルへ | 〜を、もっと美しく |
| ソリューションを提供 | お悩みに合わせたご提案をします |
| 〜にコミットする | 〜に向き合い続ける |
| シームレスな〜 | スムーズな〜 |

### 漢字→ひらがな変換（必須）

下さい→ください / 致します→いたします / 全て→すべて / 事→こと / 物→もの / 為→ため / 等→など / 頂く→いただく / 是非→ぜひ

### その他ルール

- 主語繰り返し禁止：同一セクションで同じ主語を2回以上書かない
- 誇張語制限：「最高」「究極」「完璧」は1ページ1回まで
- 「させていただく」は1ページ2回まで
- 「〜的」「〜性」は1文に1回まで
- 接続詞（また・さらに・そして）を連続して2回以上使わない
- 見出しに句点（。）を付けない
- word-break: keep-all を h1〜h3 と hero-copy のみに適用（本文・リード文には不要）
