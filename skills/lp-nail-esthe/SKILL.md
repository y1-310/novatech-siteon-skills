# lp-nail-esthe — ネイル・エステ・まつエク

## 対象
ネイルサロン、エステサロン、まつエクサロン。

## 起動条件
salon-interview から業態判定「ネイル / エステ / まつエク」で呼び出される。
ヒアリング結果（JSON）を受け取り、プラン判定→参照ファイル分岐→サイト生成。

## プラン判定→参照ファイル分岐

```
スタンダード → SKILL.md + design-system.md + sections.md
プロ         → 上記 + multipage.md
プレミアム   → 上記 + multipage.md + premium.md
```

## デザイン原則（lp-salonとの違い）

| 項目 | lp-salon | lp-nail-esthe |
|------|----------|---------------|
| 写真の比率 | 3:4縦長 | **1:1正方形** |
| ギャラリーの比重 | バランス型 | **ギャラリー最重要** |
| ビフォーアフター | オプション | **エステではほぼ必須** |
| 色傾向 | グレージュ〜ナチュラル | **フェミニン（ピンク・ラベンダー）** |

### 業態別の追加セクション

#### ネイル固有
- **デザインギャラリー**（4列グリッド、1:1正方形）
- **定額デザインコース**（月額コースの説明）
- **持ち込みデザイン案内**（Instagram等からのデザイン持ち込み）
- **オフ代・ケアメニュー**（他店オフ・自店オフの料金）

#### エステ固有
- **施術フロー図**（カウンセリング→施術→アフターケアの流れ）
- **ビフォーアフタースライダー**（施術前後の比較、ほぼ必須）
- **初回体験コースCTA**（強調表示の体験コース案内）
- **お悩み別メニュー導線**（たるみ・シミ・毛穴等→該当メニューへ誘導）

#### まつエク固有
- **デザイン比較**（ナチュラル / キュート / ゴージャス等の比較写真）
- **本数別イメージ写真**（80本 / 100本 / 120本 / 140本のイメージ）
- **素材別説明表**（シルク / ミンク / セーブル等の比較）

### ギャラリーの重要性

lp-nail-estheではギャラリーが最重要セクション。
- ネイル：デザインギャラリーが集客の要
- エステ：ビフォーアフターが説得力の要
- まつエク：デザイン比較が選択の要

ギャラリーのグリッドは **4列（デスクトップ）/ 3列（タブレット）/ 2列（モバイル）** の正方形グリッド。

### プラン別の制約

lp-salon と同一構造。SKILL.md のプラン別制約表を参照。

```
以下のルールに従ってHTMLを生成してください：
- 写真は1:1正方形を基本とする
- ギャラリーセクションを重点的に
- CSS変数を使用。ハードコードしない
- フェミニンなトーン（業態に応じて調整）
- 全imgにalt（日本語）、width、height、loading="lazy"（ヒーロー除く）
- Inter/Arial/Roboto/Helvetica禁止
```
