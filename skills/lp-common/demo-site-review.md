# デモサイトレビューチェックリスト

## デザイン
- [ ] border囲みが多用されていないか → 余白・背景色差・下線で代替
- [ ] ボタンが全ページで統一されているか（border-radius: 8px, padding: 1rem 2.5rem）
- [ ] 情報カード（営業時間・料金・電話番号）が残っていないか → デモなので削除
- [ ] 横スクロールティッカーがないか → 削除
- [ ] 写真の上にテキストが重なっていないか → 下に配置 or 削除
- [ ] Conceptセクションが1カラムか → 右に空白がないこと

## 画像
- [ ] 業態と人物の性別が一致しているか
- [ ] ギャラリーのラベルと画像が一致しているか
- [ ] AI感が強い画像がないか（§20基準）
- [ ] 写真内にテキスト・ロゴがないか

## 技術
- [ ] .nojekyll がルートにあるか
- [ ] assets/ が全て git tracked か
- [ ] 全画像が HTTP 200 か
- [ ] モバイル表示でレイアウトが崩れていないか

## テキスト
- [ ] メインコピーの改行が意図通りか（PC表示で確認）
- [ ] word-break: keep-all が日本語見出しに適用されているか
