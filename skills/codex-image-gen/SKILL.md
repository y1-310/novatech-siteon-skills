# Codex 画像生成スキル

このファイルはInstagram投稿画像を生成する前に必ず全文読むこと。

## 品質設定
quality: high を必ず指定すること。

## プロンプト形式

構造化フォーマット（Use case / Asset type 等）は使用しない。
以下の実績あるプロンプト形式に従うこと。

### テンプレート

```
Instagram投稿用カルーセル画像を{枚数}枚、個別に作成してください。
テーマ: {テーマ}
スタイル: 写実的な写真の上にダーク半透明オーバーレイ + 白テキスト
雰囲気: {雰囲気}
人物: 後ろ姿または手元のみ
配色: {配色 + ゴールドアクセントhex値}

P1（フック）:
- 背景: {P1専用の具体的な背景シーン}
- ダークオーバーレイ({透明度}%透明度)
- テキスト上部（白、小さめ）: 「{サブテキスト}」
- 中央に超大文字（白）: 「{数字}」
- 下部（ゴールド）: 「{キャッチフレーズ}」

P2（共感）:
- 背景: {P2専用の具体的な背景シーン}
- ダークオーバーレイ
- 上部（白）: 「{見出し}」
- {N}項目（白文字、ゴールドのチェックマーク）:
  ✓ {項目1}
  ✓ {項目2}
  ✓ {項目3}
- 下部（ゴールド）: 「{まとめ}」

P3（解決策）:
- 背景: {P3専用の具体的な背景シーン}
- ダークオーバーレイ
- 上部（ゴールド）: 「{見出し}」
- {N}項目（白文字、番号はゴールド）:
  01 {項目1}
  02 {項目2}
  03 {項目3}
```

## 重要ルール

### 各ページの背景は必ず異なるシーンにする
悪い例: 4枚とも「高級サロンで女性の後ろ姿」
良い例:
- P1: 暗めの高級サロン内装。間接照明。
- P2: 夜のサロン外観。ガラス越しにぼんやり光。
- P3: MacBookが開いたダークなデスク。
- P4: 施術中の手元（ぼかし強め）

### ダークオーバーレイは必ず指定する
「ダークオーバーレイ(60%透明度)」と具体的に書く。
これがないとテキストと背景のコントラストが不安定になる。

### 数字は「超大文字」と指示する
「画面を圧倒するサイズ」「大きく」等の曖昧な表現は使わない。
「中央に超大文字」が最も品質が安定する。

### 自然な日本語で書く
英語混じりの構造化フォーマット（Use case / Asset type / Constraints 等）は使わない。
プロンプトは全て自然な日本語で書く。

### テキスト色・位置は毎回明示する
- （白、小さめ）
- （ゴールド）
- （白文字、番号はゴールド）
のように、色とサイズ感を必ず括弧で指定する。

### パターンEはA/Dと別のアプローチで作る
パターンA/Dは「写真背景+カードオーバーレイ」の構成。
パターンEは「余白+タイポグラフィ主体」の構成。
パターンEにカード（白い角丸の囲み）を使ってはいけない。
色だけ変えて同じ構図にすると、雑誌感が出ない。

## パターン別の雰囲気・配色

### パターンD（ダーク高級）
雰囲気: 高級感・プロフェッショナル・シック
配色: ダークトーン（黒〜チャコール）+ 白テキスト + ゴールドアクセント(#C9A84C)
背景例: 高級サロン内装 / 夜の外観 / ダークなデスク / 施術中の手元

### パターンA（ナチュラル）
雰囲気: 温かみ・清潔感・親しみやすさ
配色: クリーム〜ベージュ + セージグリーン(#7B9E7B) + 白テキスト
背景例: 明るいカフェ空間 / 窓辺で作業する女性 / 観葉植物のあるデスク / サロン待合

### パターンE（雑誌風エディトリアル）

※ パターンEはA/Dとは根本的に異なるデザインアプローチを取る。
A/Dのテンプレート（カード+背景写真）をそのまま色だけ変えて使うのは禁止。

雰囲気: KINFOLK誌やCEREAL誌のような洗練されたミニマルデザイン
配色: 白〜ライトグレー(#F5F5F5)ベース + ネイビー(#1B3A6B)見出し + 黒(#1A1A1A)本文

構成ルール:
- テキストカード（白い角丸の囲み）は使わない。テキストは背景に直接配置
- 背景は白〜ライトグレー(#F5F5F5)。写真要素は画面の30%以下
- 見出し: ネイビー(#1B3A6B)、極太、大サイズ
- 補足文: 黒(#1A1A1A)、極細、小サイズ
- 見出しと補足文のサイズ落差を大きく取る（これが雑誌感の核）
- 区切り: ネイビー(#1B3A6B)の細い水平線（1px）
- 余白: 画面の60%以上が余白。テキストより余白が多いのが正解
- 小物: 最小限（コーヒーカップ1つ、ペン1本程度）。画面端に小さく配置

背景例: 全面ライトグレー無地 / 白い壁+小物1点 / 白い面を縦線で分割

禁止:
- 白い角丸カードで囲う（パターンAと区別がつかなくなる）
- 暖色・ベージュ・クリームの背景色
- ドライフラワー・観葉植物・木目（パターンAの要素）
- 写真が画面の50%以上を占める構図
- イラスト調の背景

## テキストルール
- 日本語誤字禁止
- 「月額9,800円〜」カンマ表記（ピリオド不可）
- 「@novatech_jp」（ipではなくjp）
- 業界略語禁止（HPB→ホットペッパービューティー等）
- ターゲット（小規模事業者）が一読で理解できる表現のみ

## 禁止事項
- 構造化プロンプト形式（Use case / Asset type 等）
- 英語混じりの指示文
- 4枚全て同じ背景・構図
- 外国人の使用
- 顔がはっきり映る構図
- イラスト・3Dレンダリング
- オーバーレイなしでテキストを直接背景に配置
