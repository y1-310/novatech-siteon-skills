# Codex 画像生成スキル

このファイルはInstagram投稿画像を生成する前に必ず全文読むこと。

## 品質設定
quality: high を必ず指定すること。

## プロンプト形式

構造化フォーマット（Use case / Asset type 等）は使用しない。
以下の実績あるプロンプト形式に従うこと。

### テンプレート

```
Instagram投稿用カルーセル画像を{枚数}枚、個別に作成してください。
```

## 重要ルール

### 各ページの背景は必ず異なるシーンにする
悪い例: 4枚とも「高級サロンで女性の後ろ姿」
良い例:
- P1: 暗めの高級サロン内装。間接照明。
- P2: 夜のサロン外観。ガラス越しにぼんやり光。
- P3: MacBookが開いたダークなデスク。
- P4: 施術中の手元（ぼかし強め）

### ダークオーバーレイは必ず指定する
「ダークオーバーレイ(60%透明度)」と具体的に書く。
これがないとテキストと背景のコントラストが不安定になる。

### 数字は「超大文字」と指示する
「画面を圧倒するサイズ」「大きく」等の曖昧な表現は使わない。
「中央に超大文字」が最も品質が安定する。

### 自然な日本語で書く
英語混じりの構造化フォーマット（Use case / Asset type / Constraints 等）は使わない。
プロンプトは全て自然な日本語で書く。

### テキスト色・位置は毎回明示する
- （白、小さめ）
- （ゴールド）
- （白文字、番号はゴールド）
のように、色とサイズ感を必ず括弧で指定する。

## パターン別の雰囲気・配色

### パターンD（ダーク高級）
雰囲気: 高級感・プロフェッショナル・シック
配色: ダークトーン（黒〜チャコール）+ 白テキスト + ゴールドアクセント(#C9A84C)
背景例: 高級サロン内装 / 夜の外観 / ダークなデスク / 施術中の手元

### パターンA（ナチュラル）
雰囲気: 温かみ・清潔感・親しみやすさ
配色: クリーム〜ベージュ + セージグリーン(#7B9E7B) + 白テキスト
背景例: 明るいカフェ空間 / 窓辺で作業する女性 / 観葉植物のあるデスク / サロン待合

### パターンE（雑誌風）
雰囲気: クリーン・ミニマル・エディトリアル
配色: 白ベース + ネイビー(#1B3A6B) + テキスト黒(#1A1A1A)
背景例: 白壁スタジオ / ミニマルなオフィス / コーヒーとノート / 明るい窓辺

## テキストルール
- 日本語誤字禁止
- 「月額9,800円〜」カンマ表記（ピリオド不可）
- 「@novatech_jp」（ipではなくjp）
- 業界略語禁止（HPB→ホットペッパービューティー等）
- ターゲット（小規模事業者）が一読で理解できる表現のみ

## 禁止事項
- 構造化プロンプト形式（Use case / Asset type 等）
- 英語混じりの指示文
- 4枚全て同じ背景・構図
- 外国人の使用
- 顔がはっきり映る構図
- イラスト・3Dレンダリング
- オーバーレイなしでテキストを直接背景に配置
