# Codex 画像生成スキル

このファイルはInstagram投稿画像を生成する前に必ず全文読むこと。

## 品質設定（必須）
画像生成時は必ず quality: high を指定すること。
デフォルト（low）では品質が大幅に劣化する。

## 構造化プロンプト形式
画像生成プロンプトは以下の形式で構造化すること:

Use case: photorealistic-natural
Asset type: Instagram carousel post
Primary request: （生成内容の説明）
Style/medium: （写真風/イラスト等）
Composition/framing: （構図の指示）
Lighting/mood: （照明・雰囲気）
Color palette: （使用色）
Text (verbatim): "（画像内テキスト一字一句正確に）"
Constraints: （必須条件）
Avoid: （禁止条件）

この形式を使うことで、Codexの画像生成の品質と一貫性が向上する。

## 生成フロー

1. このSKILL.mdを全文読む
2. P1・P2を生成 → 保存 → ユーザーに品質確認を依頼
3. 品質OKが出たら、P1・P2と同じクオリティでP3・P4を生成

## 品質統一ルール（最重要）

P3・P4の品質がP1・P2より落ちる現象が頻発する。以下を厳守:

### 4枚全てで統一する要素
- ゴールドのグロー効果の強さ（パターンDの場合）
- カード枠線のスタイル・太さ・色
- 背景写真のトーン・明度・ボケ感
- フォントの太さ・サイズ感
- 全体のコントラスト

### P3・P4生成時の追加ルール
- P1・P2の画像を必ず見返してから生成する
- テキスト要素が多くても背景の質感・光の効果を省略しない
- P1と並べて見劣りするなら再生成する

## パターンD（ダーク高級）
- 数字: 画面を圧倒するサイズ、ゴールドのグロー効果必須
- 背景: 高級サロン・バーラウンジ、間接照明、深いボケ感
- カード: 暗い半透明(rgba(0,0,0,0.75))、ゴールド枠線、角丸16px
- チェックマーク: ゴールド色
- 番号: ゴールドの丸囲み

## パターンA（ナチュラル）
- 数字: 大きく力強い、セージグリーン系
- 背景: カフェ・サロン空間、自然光、観葉植物
- カード: 白地(rgba(255,255,255,0.92))、角丸16px
- チェックマーク: セージグリーン

## パターンE（雑誌風）
- 数字: 大きく力強い、ネイビー
- 背景: 白壁スタジオ、ミニマル
- カード: 白地、薄いネイビーボーダー、角丸8px

## テキストルール
- 日本語誤字禁止
- 「月額9,800円〜」カンマ表記
- 「@novatech_jp」（ipではなくjp）
- 業界略語禁止（HPB→ホットペッパービューティー等）
- 1枚あたりテキスト要素は最大4つ

## 禁止事項
- 外国人の使用
- 顔がはっきり映る構図
- イラスト・3Dレンダリング
- P3・P4の品質をP1・P2より落とすこと
