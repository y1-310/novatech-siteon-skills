# 参謀Claude — 画像生成プロンプト作成スキル

参謀Claude（claude.ai）がCodex向けの画像生成プロンプトを作成する際のルール。
Claude Codeが指示文を作成する場合もこのルールに従うこと。

## プロンプト品質基準

### 絶対に守ること

1. プロンプトは全て自然な日本語で書く
   - 英語の構造化フォーマット禁止（Use case / Asset type / Constraints 等）
   - Codexへの指示は日本語で完結させる

2. 各ページ（P1〜P4）の背景は必ず異なるシーンにする
   - 4枚同じ構図は禁止
   - 各ページに具体的な背景描写を書く（「暗めの高級サロン内装。間接照明。」等）

3. ダークオーバーレイは透明度を数値で指定する
   - 「ダークオーバーレイ(60%透明度)」と書く
   - 「暗い半透明」等の曖昧な表現は不可

4. テキストの色・サイズ感を括弧で明示する
   - 「テキスト上部（白、小さめ）」「下部（ゴールド）」
   - 色指定なしのテキスト指示は禁止

5. 数字は「超大文字」と指示する
   - 「大きく」「圧倒するサイズ」等の曖昧な表現は使わない

6. ゴールドアクセントはhex値を添える
   - パターンD: #C9A84C
   - パターンA: #7B9E7B（セージグリーン）
   - パターンE: #1B3A6B（ネイビー）

### プロンプト構成順序

以下の順序で書くこと:

1. 全体指示（枚数・テーマ・スタイル・雰囲気・人物・配色）
2. P1の指示（背景→オーバーレイ→テキスト上から順に）
3. P2の指示（同上）
4. P3の指示（同上）
5. P4の指示（同上）
6. 出力仕様（サイズ・形式）

### パターンE専用プロンプトの注意点

パターンEはA/Dとテンプレートが異なる。以下を守ること:

1. 「テキストカード」「白い角丸カード」「rgba(255,255,255,...)」等の指示を入れない
2. 代わりに「テキストは背景に直接配置」「テキストカードは使わない」と明記する
3. 背景の指示は「白〜ライトグレー(#F5F5F5)の無地または最小限の小物」と書く
4. 「見出しは極太・大サイズ」「補足文は極細・小サイズ」とサイズ落差を指示する
5. 「ネイビー(#1B3A6B)の細い水平線（1px）で区切る」を含める
6. 「画面の60%以上が余白であること」と余白量を明示する
7. 「KINFOLKやCEREAL誌のような雰囲気」と具体的な雑誌名で方向性を示す

パターンE参照プロンプト（実績あり）:

背景: 全面ライトグレー(#F5F5F5)の無地。左下にだけコーヒーカップが小さく1つ
画面上部に小さく黒(#1A1A1A)の細字: 「{サブテキスト}」
中央にネイビー(#1B3A6B)の極太フォントで超大文字: 「{メインコピー}」
テキスト間にネイビーの細い水平線（1px、幅は文字幅の60%）
画面下部に小さく黒(#1A1A1A)の細字: 「{補足テキスト}」
全体の70%以上が余白であること

### 参照すべき実績プロンプト

以下が最も品質が高かった実績プロンプトのパターン:

```
Instagram投稿用カルーセル画像を4枚、個別に作成してください。
テーマ: {テーマ}
スタイル: 写実的な写真の上にダーク半透明オーバーレイ + 白テキスト
雰囲気: {パターン別}
人物: 後ろ姿または手元のみ
配色: {パターン別カラー + アクセントhex値}

P1（フック）:
- 背景: {P1専用の具体的シーン}
- ダークオーバーレイ(60%透明度)
- テキスト上部（白、小さめ）: 「{テキスト}」
- 中央に超大文字（白）: 「{数字}」
- 下部（ゴールド）: 「{キャッチ}」

（P2〜P4も同形式で各ページ別の背景を指定）

4枚とも1080x1080px、個別PNGで出力してください。
```

### やってはいけないこと

- 構造化フォーマットを使う（過去に品質低下を引き起こした）
- 4枚全て同じ背景にする（単調になる）
- オーバーレイの透明度を省略する（コントラスト不安定）
- テキスト色を指定しない（意図しない色になる）
- 「quality: high」をプロンプト文中に含める（スキル側で設定済み、プロンプトに混ぜると冗長）

### テキスト内容のルール

- 業界略語禁止（HPB→ホットペッパービューティー）
- 「月額9,800円〜」カンマ表記
- 「@novatech_jp」（jp）
- ターゲット（小規模事業者）が一読で理解できる表現
- キャプション1行目はフック（＼ 保存推奨 ／ 等）
